# Graphify Notes

## Purpose

Graphify generates a knowledge graph from the source code. It helps answer questions about:

- Dependency flow (which files depend on which)
- Architectural hotspots (highly connected files)
- Impact analysis (what breaks if we change X)
- Onboarding (visual map of the codebase)

## Hotspot Analysis

| Hotspot | Risk | Reason |
|---------|------|--------|
| `ClaimRepository` | High | Most complex queries, multiple joins, N+1 history |
| `PatientRepository` | Medium | Search queries across many columns |
| `AuditMiddleware` | Medium | Synchronous logging adds latency to every request |
| `PatientsController` | Medium | Orchestrates 4 repositories, single point of change |
